# Настройки безопасности Одноклассники

Одноклассники - это социальная сеть с большой аудиторией, которая часто становится целью мошенников. Взлом аккаунта может привести к краже страницы, рассылке спама и потере личных данных. Вот главные шаги для защиты.

- Минимум 12 символов
- Заглавные и строчные буквы
- Цифры
- Специальные символы (`@`, `#`, `$`, `%`, `&`)

## 2. Двухфакторная аутентификация (2FA)

Одноклассники поддерживают двухфакторную аутентификацию через приложение - аутентификатор.

**Как включить:**

1.  **Настройки → «Безопасность и VK ID» → «Двойная защита» → «SMS подтверждение»** 
2.  Нажмите **«Подключить»**
3.  Скачайте любое приложение для 2FA: Google Authenticator, Яндекс.Ключ или Authy
4.  Отсканируйте QR-код в приложении
5.  Введите код из приложения для подтверждения

Теперь при входе в аккаунт потребуется не только пароль, но и одноразовый код из вашего телефона.

## 3. Настройки приватности и активные сессии

Одноклассники собирают много данных о вас. Важно ограничить доступ к личной информации.

**Кто видит мою страницу:**
1.  **Настройки → «Приватность»**
2.  Пройдите по всем пунктам:
    - **«Кто видит мои публикации»** → поставьте «Только друзья»
    - **«Кто может комментировать мои фото/видео»** → поставьте «Только друзья»
    - **«Кто может писать мне личные сообщения»** → поставьте «Только друзья»
    - **«Кто видит мой список друзей»** → поставьте «Только я»
    
**Проверьте, откуда выполнен вход:**
1.  **Настройки → «Безопасность и VK ID» → «История посещений»** 
2.  Вы увидите список устройств и городов, откуда заходили
3.  Если видите что-то подозрительное (например, вход из другого города или с незнакомого устройства) → нажмите **«Выйти из ОК на других устройствах»**
